This is a personal project to help learn piano using a midi keyboard. The program allows you to drill many different scales and arpeggios and tracks timing, velocity, and other parameters to give you a score.

There are also features for logging "free play" time where you define a piece you're working on and can use a convenient stopwatch to time practice, or enter time manually if you've tracked it some other way.

It has features for entering a song and tracking accuracy of sectoins of the song, which you can define or modify easily.

All components are webapps that run in any modern browser using plain js/html/css.

The main components are:

Repiano: repetition tracker for piano, the main program in this project.

Repaural: experimental program to help you learn to recognize notes by ear and track accuracy.

Youloop: a program that allows you to run any youtube video (such as a piano song tutorial) and define multiple "loop sections" which you can name. You can play a loop section and vary the speed of replay.
This allows you to practice along with a video at reduced speed while learning, then incrementally increase speed as you get better.

Midilogger: this is an experimental browser based midi router that I primarily use to connect a multi-slide controller to my keyboard so it can control organ stops using midi commands.
